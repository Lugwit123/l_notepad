# L Notepad 使用说明

本地记事本 + 问 AI，笔记保存在 `notepad_list` 目录（与程序同级的文本 / Markdown / 代码文件）。

## 界面标签

| 标签 | 说明 |
|------|------|
| **主界面** | 笔记列表、编辑区、问 AI |
| **帮助** | 本页（Markdown 预览；可编辑 `help.md` 后重启生效；右键可切「Markdown 源码」） |
| **日志** | 运行过程与自动保存等提示 |

## 笔记

- **新建 / 保存 / 删除**：底部按钮；修改后标题栏旁会标记未保存。
- **搜索**：左侧「搜索标题…」按标题过滤。
- **置顶/收藏**：将当前笔记加入收藏并排在列表前。
- **拖拽排序**：在列表内拖动条目调整顺序。
- **打开文件**：编辑外部文件（保存写回原路径）。
- **双击标题**：重命名笔记文件。
- **自动保存**：窗口失去焦点、最小化时自动保存（最小化时托盘可提示）。

## 问 AI

1. 左侧点 **「问AI / 新建对话」** 进入 AI 模式。
2. 在 `notepad_list` 下需有对应 **`问AI*.md`** 文件才会恢复该会话标签（如 `问AI 1.md`）。
3. 选择 **模型** → 在输入区提问 → **问AI**。
4. 支持多标签会话；标签可关闭；会话与草稿会写入本地设置。

需配置环境变量 **SiliconFlow API Key** 方可请求模型。

## 编辑器

- **高亮模式**（主界面）：普通文本 / Python / Markdown / 日志等，按扩展名也可自动切换。
- **Ctrl + 滚轮**：调整编辑区与日志字体大小（8–28）。
- **粘贴图片**（笔记编辑区）：可保存到 `notepad_list` 并插入 Markdown 图片链接。

## 全局快捷键（Windows）

- **双击热键**：从任意位置唤出 / 置前窗口（按键和间隔可在 **快捷键** 按钮调节，支持 Ctrl/Alt/Shift，间隔约 0.05–1.0 秒）。

## 系统托盘

- 关闭窗口默认 **最小化到托盘**（非退出）。
- 托盘菜单：**显示/恢复**、**最小化到托盘**、**退出**。
- 双击托盘图标恢复窗口。

## 数据位置

- 笔记目录：`<l_notepad>/notepad_list/`
- 界面与 AI 会话：`QSettings`（组织名 `Lugwit`，应用名 `l_notepad`）

## 重启

主界面 **重启** 按钮会重新启动当前进程（用于加载代码或配置变更后）。
